Presentation

Introduction

  • A web app developed in django which analyzes a research paper and represents it in form of HTML page for faster understanding as well as provides a JSON downloadable file.
  • Users needs to upload a standard pdf file and let server handle all the tricky stuff!

Tools and Technologies

  • Web Application - Django
  • Natural Language Processing - NLTK
  • Logistic Regression - Classifier
  • Neural Network - Spacy
  • Dataset - en_core_web_lg,polarity_senttoken

Title Prediction

  • Convert the corpus into sentence using sent_tokenize
  • Convert the sentences into words using word_tokenize
  • Maintain the dictionary of frequency of all words
  • Calculate the score of first 5 percentage of sentence in corpus
  • Sort those sentences in descending order
  • To get a multi-line title : if the difference between first and second sentence is less than some difference or if the count of last word in first sentence and first word of second sentence is greater than a specific limit

Authors Extraction

  • Taking 20 percent of data as corpus and using spacy we can get a basic idea of all the proper names in corpus
  • Spacy provides 3 types of dataset : en_core_web_sm,en_core_web_md,en_core_web_lg
  • Using name_entity function user can get idea of all the proper names in corpus

Reference Extraction

  • Find all the occurrence of word "References"
  • Make a dummy data of approx 40 lines for each occurrence of word "reference"
  • Count number of proper names in each dummy data
  • One having maximum amount proper names will be selected as index of reference

Problem Statement

  • Using data-set (polarity_sentoken) make a classifier through Logistic regression
  • Generate pickle files and import pickle files in current working directory
  • Perform semantic analysis on the corpus and generate top 5 percent negative sentance

    • Key Points and KeyWords

    • Convert the corpus into sentence using sent_tokenize
    • Convert the sentences into words using word_tokenize
    • Maintain the dictionary of frequency of all words
    • Sort the sentences in decreasing order of their weights by this we can get a list of all key points
    • Sort the words in descending order of their frequencies to get the list key words which are then used to plot the word-frequency bar graph

    Abstract Finder

    • TO extract an abstract from research paper, we have analysed all words of research paper and then we select a paragraph which has more frequent words
    • Thus by using natural language processing, we extract an abstract from research paper

    Limitations

    • Limited to only English data-set
